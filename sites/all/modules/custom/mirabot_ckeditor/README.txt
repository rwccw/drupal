Mirabot CKeditor
A Drupal feature for easier configuration of the CKeditor WYSIWYG editor.

-- About

Mirabot CKeditor uses the Drupal contrib module Features to package up some
basic CKeditor configuration for the Filtered HTML text format.

-- Prerequisites

1. features (https://drupal.org/project/features)
2. ckeditor (https://drupal.org/project/ckeditor)

-- Installation Instructions

1. Install and enable the features and ckeditor modules.
2. Install the ckeditor library to the sites/all/libraries directory. See the
ckeditor instructions for more information.
3. Enable mirabot_ckeditor.
4. If necessary, revert overwritten components in the Mirabot CKeditor
feature.
